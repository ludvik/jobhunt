# Base Resume Classification Prompt

Given a **Job Description (JD)**, choose the best base resume direction from:

- **ai** — AI/ML engineering roles (model training, inference, ML infrastructure)
- **ic** — General senior IC roles (backend, platform, distributed systems)
- **mgmt** — Engineering management / director roles (team building, strategy, delivery)
- **venture** — Venture builder / 0-to-1 roles (startup, product engineering, full-stack)

## Output Format

Output ONLY one word: `ai`, `ic`, `mgmt`, or `venture`. Nothing else.
